Escolha uma das opcoes para gerar o aplicativo.


opcoes:

  • 1-Modo rapido.
  • 2-Modo passo a passo.

  • Modo Rapido ---->

    Sera gerado aplicativo com o nome do "banco de dados" e configuracoes basicas para a grid e para os componentes do formularios do crud.

  • Modo passo a passo ---->

    Sera solicitado que o usuario defina as configuracoes da GRID e validacoes para os componentes do formulario para cada Tabela.

    • Se o usuario escolher o modo rapido as configuracoes da GRID e as validacoes dos componentes dos formularios poderao ser configuradas manualmente basta editar os arquivos ".json" que serao gerados na pasta "./php/json" do modulo gerado.
