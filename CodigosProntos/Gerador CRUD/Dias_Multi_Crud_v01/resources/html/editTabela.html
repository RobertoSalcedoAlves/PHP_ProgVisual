Para editar a tabela

Opcoes:

  • 1-Utilize a grid acima para editar a tabela.
  • 2-Clique em 'adicionar um campo' para criar uma linha na grid.
  • 3-De dois cliques nas celulas da grid para editar.
  • 4-Edite a linha da grid com os dados do campo que deseja criar.
  • 5-Clique em 'Excluir um campo' para excluir um campo.
  • 6-No final Clique em 'Gravar alteracoes' para salvar as alteracoes no MySql.
