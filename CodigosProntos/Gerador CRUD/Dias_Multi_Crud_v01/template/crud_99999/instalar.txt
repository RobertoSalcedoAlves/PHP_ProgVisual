Para instalar -> basta descompactar os arquivos no seu Wamp ou na pasta www do seu servidor.

No arquivo "tabela.sql" tem a sql para criar a tabela que foi utilizada no gerador crud.

No Arquivo "Config.php" coloque os dados da conexao mysql
No arquivo "Config.json" estão as configuracoes do formulario (NAO ALTERE estas configuracoes 
foi gerada pelo gerador crud de acordo com as escolhas na base mysql).

Os arquivos "Config.json" e "Config.php" são os que contém as configurações 
do crud gerados a partir das escolhas na base da dos Mysql os demais arquivos são cópias, 
ou seja, são iguais em todos os CRUD Gerados no aplicativo Gerador Crud.

O aplicativo Gerador crud gera apenas as configurações do crud.
As configurações geradas estão nos arquivos:
Config.php -> Dados para conexao e consulta sql.
Config.json -> Dados para montar o formulario.
Voce pode alterar essas configurações como quizer, mas veja que os dois arquivos tem 
dados que dependem um do outro para montar a GRID e o FORMULARIO.
